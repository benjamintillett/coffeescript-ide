
Coffescript Ide 
===============

@ Makers Academy
----------------

An incomplete ide built using node.js coffescript mocha and zombie
Skills learnt/used
-----------------
> * Javascript 
> * Jquery
> * mocha 
> * zombie
> * socket.io


How to use
----------

Clone the repository:

```shell 
$ git clone git@github.com:benjamintillett/coffeescript-ide.git

```

run tests in mocha:

```shell 
$ cd coffeescript-ide
$ mocha
```

Start the server:

```shell 
$ npm start
```

Point your browser to http://localhost:3000/ to use
